FAQ
What is Predictiveai.tech?
Predictiveai.tech is a WebApp for Predictive Analytics. Currrently providing a service called DART(Data art) which gives a GUI interface for Data Analyse, Pre-process ,Visualize and use some basic algorithms over the Data for Prediction.
What type of file can i upload?
You can upload files with .csv, .xlsx or .json formats.
What is Target column?
The target column contain whatever the output of the input variables. It could be the individual classes that the input variables maybe mapped to in case of a classification problem or the output value range in a regression problem.
What is Primary column?
A primary column is a column of table which uniquely identifies each tuple (row) in that table.
What is Categorical column?
Categorical column represent types of data which may be divided into groups. Examples of categorical variables are race, sex, age group, and educational level.
What is EDA?
Exploratory data analysis (EDA) is an approach to analyzing data sets to summarize their main characteristics, often with visual methods.
